## Browser Wallets

When interacting with the Realms World L3 via a browser wallet you will need to add the network to your wallet. The network details are as follows:



#### Your Argent should look like this:

```
Network Name: Realms World L3
Chain ID: DOJO_REALMS_WORLD
RPC URL: https://api.cartridge.gg/x/realms/katana

Account Class hash: 0x029927c8af6bccf3f6fda035981e765a7bdbf18a2dc0d630494f8758aa908e2b
Fee Token Address: 0x51205c5e6ac3ad5691c28c0c5ffcdd62c70bddb63612f75a4bac9b2a85b9449
Multicall Address: 0x51205c5e6ac3ad5691c28c0c5ffcdd62c70bddb63612f75a4bac9b2a85b9449
```
